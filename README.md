  –ú–∏–∫—Ä–æ—Å–µ—Ä–≤–∏—Å–Ω–∞—è –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞ - –¥–∏–∑–∞–π–Ω –ø–∞—Ç–µ—Ä–Ω –ø—Ä–æ–µ–∫—Ç–∏—Ä–æ–≤–∞–Ω–∏—è –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–π, –ø—Ä–∏ –∫–æ—Ç–æ—Ä–æ–º –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –Ω–∞–±–æ—Ä –Ω–µ–∑–∞–≤–∏—Å–∏–≤–æ —Ä–∞–∑–≤—ë—Ä–Ω—É—Ç—ã—Ö —Å–µ—Ä–≤–∏—Å–æ–≤. –ß–∞—â–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ø—Ä–æ–µ–∫—Ç–∏—Ä–æ–≤–∞–Ω–∏—è —Å–ª–æ–∂–Ω—ã—Ö "enterprise" –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–π.

  –•–∞—Ä–∞–∫—Ç–µ—Ä–Ω—ã–µ –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏:
  
  1) –ö–æ–º–ø–æ–Ω–µ–Ω—Ç—ã = —Å–µ—Ä–≤–∏—Å—ã (–≤ –ø—Ä–æ—Ç–∏–≤–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –±–∏–±–ª–∏–æ—Ç–µ–∫–∞–º) </br>
     –° —Ç–æ—á–∫–∏ –∑—Ä–µ–Ω–∏—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–Ω–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞, —Å–µ—Ä–≤–∏—Å—ã - —ç—Ç–æ –≤–Ω–µ—à–Ω–∏–µ –ø–æ –æ—Ç–Ω–æ—à–µ–Ω–∏—é –∫ —Ç–µ–∫—É—â–µ–º—É –ø—Ä–æ—Ü–µ—Å—Å—É –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –æ–±—â–∞—é—Ç—Å—è –ø–æ—Å—Ä–µ–¥—Å—Ç–≤–æ–º –∑–∞–ø—Ä–æ—Å–æ–≤ –æ—Ç –≤–µ–±-—Å–µ—Ä–≤–∏—Å–æ–≤ –∏–ª–∏ RPC (Remote Procedure Call).
     
  2) Independent deployability </br>
     –°–µ—Ä–≤–∏—Å—ã —Ä–∞–∑–≤–æ—Ä–∞—á–∏–≤–∞—é—Ç—Å—è –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ –¥—Ä—É–≥ –æ—Ç –¥—Ä—É–≥–∞ (—Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ —Ä–µ–ª–∏–∑–∏—Ç—å—Å—è —Ç–æ–∂–µ –º–æ–≥—É—Ç –æ—Ç–¥–µ–ª—å–Ω–æ). –ù—É–∂–Ω–æ —Å—Ç—Ä–µ–º–∏—Ç—Å—è –∫ —Ç–æ–º—É —á—Ç–æ–±—ã —Å–µ—Ä–≤–∏—Å—ã –±—ã–ª–∏ –º–µ–Ω—å—à–µ —Å–≤—è–∑–∞–Ω—ã –¥—Ä—É–≥ —Å –¥—Ä—É–≥–æ–º (decoupled), —á—Ç–æ–±—ã –ø—Ä–∏ –∏–∑–º–µ–Ω–µ–Ω–∏–∏ –æ–¥–Ω–æ–≥–æ —Å–µ—Ä–≤–∏—Å–∞ –Ω–µ –±—ã–ª–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –º–µ–Ω—è—Ç—å —á—Ç–æ-–ª–∏–±–æ –µ—â—ë. 
  
  2) –°–µ—Ä–≤–∏—Å—ã –æ—Ä–≥–∞–Ω–∏–∑–æ–≤–∞–Ω—ã –ø–æ –ø—Ä–∏–Ω—Ü–∏–ø—É –±–∏–∑–Ω–µ—Å-–¥–æ–º–µ–Ω–æ–≤ (domain-driven design) </br>
     –≠—Ç–æ –ø–æ–º–æ–≥–∞–µ—Ç —Ä–∞—Å—Å—Ç–∞–≤–ª—è—Ç—å –≥—Ä–∞–Ω–∏—Ü—ã –º–µ–∂–¥—É —Å–µ—Ä–≤–∏—Å–∞–º–∏ –∏ –≤ —Ü–µ–ª–æ–º —Ä–∞–∑–±–∏–≤–∞—Ç—å –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ –Ω–∞ –æ—Ç–¥–µ–ª—å–Ω—ã–µ —Å–µ—Ä–≤–∏—Å—ã.

     –ù–µ–∑–∞–≤–∏—Å–∏–º—ã–µ —Å–µ—Ä–≤–∏—Å—ã = smart endpoints - –ø—Ä–∏–Ω–∏–º–∞—é—Ç –∑–∞–ø—Ä–æ—Å—ã, –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—é—Ç –∏—Ö —Å–æ–≥–ª–∞—Å–Ω–æ –±–∞–∑–Ω–µ—Å –ª–æ–≥–∏–∫–µ –∏ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –æ—Ç–≤–µ—Ç. –°–ª–æ–∂–Ω–∞—è –≤–Ω—É—Ç—Ä–µ–Ω–Ω—è—è –∏–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—è —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª–∞ –¥–æ–ª–∂–Ω–∞ –±—ã—Ç—å —Å–∫—Ä—ã—Ç–∞ (Hexagonal pattern) –æ—Ç –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–æ–≤, —á–µ—Ä–µ–∑ –∫–æ—Ç–æ—Ä—ã–µ –æ—Å—É—â–µ—Å—Ç–≤–ª—è–µ—Ç—Å—è –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏–µ —Å —ç—Ç–∏–º —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª–æ–º.
